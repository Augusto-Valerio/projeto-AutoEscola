# AutoEscola_System
O AutoEscola System é um sistema desenvolvido em Java com o objetivo de simular a gestão de uma autoescola. A aplicação permite o cadastro de alunos, instrutores e veículos, além de possibilitar o agendamento e cancelamento de aulas teóricas e práticas, e a realização de exames para aprovação dos alunos.
